About This Website

This website was built as a clean and simple directory for displaying large collections of numbered cards.

Users can quickly search, filter, and open links directly from the cards.

The design is lightweight, responsive, and works perfectly with GitHub Pages, making it easy to host completely for free.

You can customize the cards, categories, colors, and links however you want.
